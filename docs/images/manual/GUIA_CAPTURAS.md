# Guia de Capturas de Pantalla para el Manual de Usuario

## Instrucciones

Para completar el manual de usuario, debe tomar las siguientes capturas de pantalla del sistema en funcionamiento y guardarlas en esta carpeta con los nombres indicados.

**Formato recomendado:** PNG
**Resolucion recomendada:** 1920x1080 o superior
**Herramientas sugeridas:** Snipping Tool (Windows), Screenshot (Mac), o extensiones de navegador

## Notas Adicionales

1. **Consistencia:** Mantenga el mismo tamano de ventana en todas las capturas
2. **Datos de prueba:** Use datos de prueba representativos pero no reales
3. **Ocultar datos sensibles:** Difumine o reemplace datos confidenciales
4. **Resaltados:** Puede agregar flechas o circulos para senalar elementos importantes
